    Interface StackRegistryEntry

    Registry entry representing an installed stack and its components

    Remarks

    This interface tracks all information about a stack installation including its metadata, installation source, version information, and detailed component mapping. Used by StackRegistryService to manage stack lifecycles and enable proper uninstallation, conflict detection, and component tracking.

    Example 

    // Registry entry tracks all installed components for a stack
    const entry = await registryService.getStackEntry('myorg/awesome-stack');
    if (entry) {
      console.log('Stack name:', entry.name);
      console.log('Commands installed:', entry.components.commands.length);
    }
