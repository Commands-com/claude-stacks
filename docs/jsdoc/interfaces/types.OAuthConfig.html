    Interface OAuthConfig

    OAuth 2.0 configuration for Commands.com authentication

    Configuration parameters for OAuth 2.0 authentication flow with Commands.com. Used by the authentication system to handle user login, token exchange, and authorization for publishing and managing stacks on the platform.

    Example 

    const oauthConfig: OAuthConfig = {
      clientId: 'claude-stacks-cli',
      authUrl: 'https://backend.commands.com/oauth/authorize',
      tokenUrl: 'https://backend.commands.com/oauth/token',
      redirectPort: 8080
    };

    OAuth authorization endpoint URL
