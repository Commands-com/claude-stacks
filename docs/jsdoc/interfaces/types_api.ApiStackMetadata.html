    Interface ApiStackMetadata

    Stack metadata from Commands.com API responses

    Contains timestamps, versioning information, and installation details for stacks. This metadata tracks the lifecycle of a stack from creation through installation and updates, providing traceability for stack management operations.

    Example 

    const metadata: ApiStackMetadata = {
      created_at: '2024-01-15T10:30:00Z',
      updated_at: '2024-01-20T14:45:00Z',
      published_version: '1.2.0',
      local_version: '1.2.0',
      installed_from: 'commands.com/example/my-stack',
      installed_at: '2024-01-21T09:15:00Z',
      cli_version: '1.4.2'
    };
