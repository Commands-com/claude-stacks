    Class MetadataService

    Service for stack metadata operations

    Remarks

    Encapsulates metadata utilities to reduce coupling in action layer. Provides a clean interface for managing published stack metadata including creation, retrieval, updates, and deletion.

    findStackByStackId

    • findStackByStackId(
          stackId: string,
      ): Promise<null | { metadata: PublishedStackMetadata; projectPath: string }>

      Find a stack by its stack ID across all published stack metadata

      Searches through all locally stored published stack metadata to find a stack with the specified stack ID. This is useful for stack management operations that need to locate a stack's project directory and metadata based on its unique identifier.

      Parameters

      • stackId: string

        The unique stack identifier to search for (format: org/name)

      Returns Promise<null | { metadata: PublishedStackMetadata; projectPath: string }>

      Promise that resolves to an object containing the project path and metadata, or null if no stack with the given ID is found

      Throws

      If metadata file cannot be read or is corrupted

      Example 

      const metadataService = new MetadataService();
      const result = await metadataService.findStackByStackId('myorg/mystack');
      if (result) {
        console.log(`Found stack at: ${result.projectPath}`);
        console.log(`Last published version: ${result.metadata.last_published_version}`);
      } else {
        console.log('Stack not found');
      }

    generateSuggestedVersion

    • generateSuggestedVersion(lastVersion: string): string

      Generate a suggested next version for a stack based on the last published version

      Automatically increments the patch version of the provided semver string to suggest the next logical version for publication. Follows semantic versioning conventions for patch-level increments.

      Parameters

      • lastVersion: string

        The last published version in semver format (e.g., '1.0.0')

      Returns string

      The suggested next version with patch incremented (e.g., '1.0.1')

      Throws

      If the lastVersion is not a valid semver format

      Example 

      const metadataService = new MetadataService();
      const suggested = metadataService.generateSuggestedVersion('1.0.0');
      console.log(suggested); // '1.0.1'

      Get all published stack metadata from the local metadata store

      Retrieves the complete collection of metadata for all stacks that have been published from this machine. The returned record maps project paths to their corresponding published metadata.
